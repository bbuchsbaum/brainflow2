# Brainflow Documentation Archive

This archive contains the original Phase 1 planning and design documents for the Brainflow project. These documents have been consolidated into the parent directory's blueprint and status files for easier reference.

## Archive Structure

### phase1-planning/
Contains all original planning documents from the initial WebGPU v2 architecture design phase:

- **ADR-xxx**: Architecture Decision Records
- **PLAN-xxx**: Development plans and milestones  
- **GUIDE-xxx**: Implementation guides
- **Sprint-x**: Sprint backlogs and task breakdowns
- **DEV-xxx**: Developer guides for setup and testing
- Other supporting documents

## Why Archived?

These documents served their purpose during the planning phase but became unwieldy for day-to-day development reference. The consolidated files in the parent directory provide:

- **Project_Blueprint.md**: Technical architecture and guidelines
- **Project_Status.md**: Current progress and active work

The archived documents remain available for:
- Historical reference
- Detailed specifications when needed
- Understanding design decisions
- Audit trail of project evolution
